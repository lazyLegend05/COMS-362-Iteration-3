# COMS 3620: Hospital Team Project
## Iteration 01

**Introduction:** 
Our project focuses on developing a Hospital Management System that manages the core operations of a hospital across multiple departments. The system will handle key processes such as emergency patient registration, pharmacy management, laboratory services, and medical imaging. Each team member is responsible for one department. Together these departments cover the essential day to day functions of a hospital, from admitting and treating patients to dispensing medication and processing diagnostic results.
